# ASP.NET Core - MongoDB Repository & Unit of Work Pattern

Example App - This demo shows how to simple implement repository and unit of work patterns for MongoDB .Net driver.

Pre reqs

* Net.Core 3.1

Techs:

* ASP.NET Core
* Swagger
* ServiceStack.Core

1. Download code

  * Open a terminal
  * docker-compose up --build
